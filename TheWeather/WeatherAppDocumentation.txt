WeatherApp

WeatherApp - это простое JavaFX-приложение для отображения текущей погоды в выбранном городе. Оно использует API OpenWeatherMap для получения метеорологических данных.

Возможности
Ввод названия города
Получение актуальной информации о погоде:
Температура
Ощущаемая температура
Максимальная и минимальная температура
Давление
Описание погоды (облачно, ясно и т. д.)
Графический интерфейс с оформлением
Установка иконки приложения

Требования
Java 8+
JavaFX (включён в JDK 11+, для JDK 8 требуется дополнительная настройка)
API-ключ OpenWeatherMap
